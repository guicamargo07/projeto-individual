# Sobre o projeto

É um website temático dedicado à trajetória de Cristiano Ronaldo. O projeto foi criado com o objetivo de reunir informações sobre a carreira, conquistas e estatísticas de um dos maiores jogadores da história do futebol, ao mesmo tempo em que aplica os conhecimentos adquiridos durante o semestre.

# Objetivo

O principal objetivo do projeto é aplicar, de forma prática, os conceitos estudados ao longo do semestre no curso de Análise e Desenvolvimento de Sistemas, desenvolvendo uma aplicação Full Stack que integra Front-end, Back-end e banco de dados. Além disso, o projeto busca unir a aprendizagem técnica à paixão pelo futebol e pela trajetória de Cristiano Ronaldo.

# Funcionalidades

- Cadastro de usuários
- Login e autenticação
- Sistema de login e logout
- Dashboard com estatísticas e indicadores sobre Cristiano Ronaldo
- Exibição de informações sobre a carreira e conquistas do jogador
- Integração com banco de dados para armazenamento e consulta de informações
- Interface responsiva e intuitiva
